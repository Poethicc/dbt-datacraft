## Список DAG’ов

| Название               | Рещаемая задача                                                                                                                  |
| ---------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| [[create_connections]] | Создаёт Airbyte Connection                                                                                                       |

| [[sync_data]]          | Синхронизирует соединения в Airbyte и запускает модели dbt                                                                       |
